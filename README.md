# Todo List - Список задач

Простое приложение для управления задачами, созданное с использованием React, TypeScript и SCSS модулей.

## Запуск

```bash
# Клонирование репозитория
git clone https://github.com/Artland13/todo-test.git

# Переход в папку проекта
cd todo-list

# Установка зависимостей
npm i

# Запуск в режиме разработки
npm run dev
```
